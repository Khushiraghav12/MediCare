# MedTech Frontend

A modern healthcare platform frontend built with React, featuring AI-powered medical report analysis, intelligent chat assistance, and doctor location services.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm start
   ```

3. **Open in browser:**
   Navigate to `http://localhost:3000`

## Features

- 🏠 **Homepage** - Modern landing page with animations
- 🔐 **Login** - Authentication with social options
- 📄 **Upload Reports** - 5 medical report types
- 🤖 **GARUN.ai Chat** - AI-powered health assistant
- 🏥 **Find Doctors** - Geolocation-based doctor search

## Tech Stack

- React 18
- React Router DOM
- Framer Motion
- Lucide React Icons
- CSS3 with animations
